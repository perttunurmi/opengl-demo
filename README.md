## Building

### Nix (preferred)

Make sure you have [flakes and the nix-command](https://nixos.wiki/wiki/Flakes#Enable_flakes_temporarily) enabled.

The file [flake.nix](./flake.nix) will take care of the rest.

Build the project with

```
nix build
```

or

Build and run with

```
nix run
```
